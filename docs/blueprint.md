# **App Name**: BaristaTrack

## Core Features:

- Order Input: Allow baristas to input new customer orders, specifying drinks and any customizations.
- Order Tracking: Display orders in a clear, real-time list, showing the status of each order (e.g., 'pending', 'in progress', 'ready').
- Status Updates: Enable baristas to easily update the status of each order as it moves through the preparation process.
- AI Priority Reordering Tool: An AI-powered tool analyzes current orders, prep times, and ingredients to suggest optimal order preparation sequence, minimizing wait times.

## Style Guidelines:

- Primary color: Deep brown (#4E342E), reminiscent of coffee.
- Background color: Light beige (#F5F5DC) to provide a warm and inviting feel.
- Accent color: Burnt orange (#CC6633) for interactive elements and notifications.
- Body and headline font: 'PT Sans', a humanist sans-serif for clarity and warmth.
- Use simple, clean icons to represent different drink types and order statuses.
- Prioritize a clear, uncluttered layout for quick scanning and efficient use in a busy bar environment.
- Subtle animations to indicate order status changes and provide visual feedback on interactions.
